# TaskCanvas

A comprehensive task management application.

## Features
- Unified Schedule View with Time Blocks, Tasks, and Habits
- Inspiration Hub with quotes, videos, music, images, and Bible verses
- Smart Lists filtering
- Mobile support

## Tech Stack
- Frontend: React 18, TypeScript, TanStack Query, Tailwind CSS
- Backend: Node.js, Express.js
- Database: PostgreSQL with Drizzle ORM
